# Founder OS Vault

## Purpose

This vault stores the operating memory for Web Studio OS.

## Main Areas

- `10-business/` business context, strategy and decisions
- `20-clients/` ICP, pains and client patterns
- `30-offers/` offer design and launch constraints
- `40-seo-market/` demand capture and content ideas
- `50-sales-mail/` tone, objections and outreach rules
- `60-admin/` permissions, admin policy and operating rules
- `70-learning/` skills to build and recurring lessons
- `90-agent-runs/` journals and execution traces

## Coach Rule

The Coach agent must cite the notes it uses and avoid answering beyond the vault content.

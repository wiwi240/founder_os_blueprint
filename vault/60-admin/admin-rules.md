# Admin Rules

## Operating Rules

- every quote is a draft until human validation
- every client promise needs written scope assumptions
- keep a dated record of decisions and runs
- store reusable process knowledge in the vault

## Risks

- vague scope becomes unpaid work
- missing assumptions create delivery conflict
- undocumented decisions make follow-up weak

## Coach Signals

- learn lightweight quoting discipline
- learn handoff discipline between sales and delivery
- learn what must be written down every time

# Core Offer

## Main Offer

Fast landing page or brochure site for local businesses with a short scoping phase and a standard delivery process.

## Packaging

- landing page around a fixed frame
- brochure site with controlled page count
- optional local SEO setup
- optional maintenance plan

## Launch Constraints

- no fixed quote before scoping inputs are confirmed
- no heavy design exploration in the MVP offer
- keep onboarding lightweight and repeatable

## Coach Signals

- learn how to productize the offer
- learn how to define a clean scope boundary
- learn how to present options without creating confusion
